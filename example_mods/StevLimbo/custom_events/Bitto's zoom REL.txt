value 1 sets the zoom: in this version, its RELATIVE to the stage's original zoom. (multiplicative)
to go back to the original zoom, input 1. 

This event automatically switches to custom mode when value 2 is used. leave blank to use the regular mode.
value 2 is the zoom duration (in beats), and the tween used.
